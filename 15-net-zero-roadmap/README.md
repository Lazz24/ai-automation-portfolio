# 15 — Net Zero Roadmap Generator

An AI-powered tool that generates a structured Net Zero roadmap from a simple company profile form. Outputs an executive summary, milestone timeline, quick wins, key risks, and a downloadable branded PDF report.

## Stack
- Groq API (llama-3.3-70b-versatile)
- Cloudflare Workers (API proxy + secret vault)
- Supabase (Postgres database)
- jsPDF (client-side PDF generation)
- GitHub Pages (static hosting)
